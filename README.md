# TinyGameMachine

An Engineer Cafe workshop starts from May 17th 2026.

## 特徴 (Features)

- MCU: Seeed Studio XIAO ESP32C3 (https://wiki.seeedstudio.com/ja/XIAO_ESP32C3_Getting_Started/)
- Display: 0.96" OLED 128 x 64 pix SSD1306
- Light: 3 NeoPixels
- Sound: Passive Buzzer
- Control: 4 momentary switches
